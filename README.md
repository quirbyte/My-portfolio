A high-performance, responsive portfolio website built to showcase a journey from low-level systems logic to modern full-stack architecture. This project features advanced animations, a dynamic contact system, and a clean, "Space-Modern" aesthetic.

## 🚀 Technical Highlights

* **Engineering Foundation:** Architecting scalable systems and high-performance logic, bridging the gap between foundational data structures and modern web frameworks.
* **MERN Architecture:** Leveraging **MongoDB**, **Express**, **React**, and **Node.js** for robust, end-to-end application development.
* **TypeScript Integration:** Ensuring type-safe, maintainable code across the entire frontend ecosystem.
* **Dynamic UX:** Utilizing **Framer Motion** for physics-based animations and **Tailwind CSS** for a mobile-first, glassmorphic design.
* **Automated Communication:** Integrated **EmailJS** for direct client-to-developer messaging without a dedicated backend server.

## 🛠️ Tech Stack

* **Frontend:** React 18, Vite, TypeScript, Tailwind CSS
* **Animations:** Framer Motion, React-Icons
* **Tools:** EmailJS, Particle-Background components
* **Core Logic:** C, C++, Java, Python, MySQL
