A high-performance, responsive portfolio website built to showcase modern web development capabilities. This project features physics-based animations, a dynamic contact system, and a clean, "Space-Modern" aesthetic designed for an optimal user experience.

## 🚀 Technical Highlights

* **Full-Stack Architecture:** Engineering end-to-end web solutions using the MERN ecosystem for seamless data flow and performance.
* **Dynamic UX:** Utilizing **Framer Motion** for interactive, physics-driven animations and **Tailwind CSS** for a mobile-first, glassmorphic design.
* **Automated Communication:** Integrated **EmailJS** for direct client-to-developer messaging, featuring real-time form validation and status handling.
* **Responsive Design:** Architecting scalable layouts that provide a consistent experience across all device types and screen sizes.

## 🛠️ Tech Stack

* **Frontend:** React 18, Vite, JavaScript, Tailwind CSS
* **Backend Logic:** Node.js, Express
* **Animations:** Framer Motion, React-Icons
* **Tools & Services:** EmailJS, Particle-Background components
